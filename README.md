## Description :bulb:
This repository contains a collection of interview preparation materials and practice problems for Holberton School students. It covers various aspects of technical interviews, including:

* **Whiteboard**
* **Data Structures**
* **Time complexity (Big O notation)**
* **Searching algorithms**

The problems are designed to enhance algorithmic thinking and problem-solving skills, crucial for technical interviews in the software industry.
